# Contributing to OISS

Thank you for your interest in contributing to the Open Internet Sharing System (OISS)!

## How to Contribute
1. **Fork the Repository:** Create your own fork and clone it to your local machine.
2. **Create a Branch:** Always create a feature branch (`git checkout -b feature/your-feature-name`).
3. **Make Changes:** Write clean, well-documented code. Follow Flutter lint rules and Python PEP8 standards.
4. **Test:** Verify your changes do not break existing privacy controls. Run the backend and frontend locally.
5. **Submit a Pull Request (PR):** Describe your changes in detail in the PR description. Link to any relevant issues.

## Reporting Bugs
- Use the GitHub Issue Tracker to report bugs.
- Provide a clear and descriptive title.
- Explain the steps to reproduce the issue.
- DO NOT report security vulnerabilities here. Check `SECURITY.md`.

## Core Principles
OISS is built on privacy. We NEVER log IP addresses. Any PR that introduces logging of PII (Personally Identifiable Information), real IP addresses, or permanent tracking will be immediately rejected.

Thank you for helping keep the internet open and safe!
